## 介绍
“组合4K”技术能够将4块1920×1080的画面拼合为1块3840×2160（也就是4K）的画面。
搭配上“IPTV多线程”技术，每个1920×1080的画面为一个线程。
这样可以保证信号安全，减少“彁”信号和其它信号劫持的风险。
